## 🌟 Overview
**WhatsApp Bot MultiDevice** is a highly efficient, fast, and secure automation solution built for WhatsApp. Designed to simplify group management, provide smart responses, and offer entertainment tools.

## ✨ Key Features
* 💬 **Smart Chatbot (Eva):** Natural conversations and query responses.
* 📸 **Multimedia Mastery:** Convert images/videos to stickers.
* 🛡️ **Advanced Security:** Google OAuth for admin panels.
* 🚀 **Multi-Device Support:** No phone connection required.
* 📊 **Live Dashboard:** Monitor bot logs in real-time.

## 🚀 Deployment Guide
1. Fork the Repository.
2. Click the Deploy to Render button.
3. Configure Environment Variables: SESSION_SECRET, BOT_NUMBER, MY_NUMBER, PREFIX.
